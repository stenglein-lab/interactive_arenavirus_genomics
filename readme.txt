To view interactive figures:

(1) unpack the contents of this tar file into some directory
(2) cd to that directory
(3) run ./run_server [runs python simple web server]
(4) open a browser and open either of the following urls:

http://localhost:8888/cophylogeny.html            ---> interactive co-phylogeny for S and L segments
http://localhost:8888/segment_cartoons.html       ---> genome segment cartoons including coverage and 2Ëš structure plots
